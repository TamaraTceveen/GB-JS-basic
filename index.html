Задание 1

var a = 1, b = 1, c, d;

c = ++a; alert(c); // 2

в переменной с хранится число 2, так как сначала идет операция увеличения переменной а на единицу, а уже потом происходит присвоение значения переменной с.
d = b++; alert(d); // 1

вывод цифры 1: сначала происходит присвоение значения переменной d, после этого переменная b увеличивается на единицу. Таким образом d = 1; b = 2;
c = (2+ ++a); alert(c); // 5

вывод 5: переменная а в настоящий момент содержит 2. Затем мы увеличиваем ее еще на 1, после этого прибавляем 2 (2 + (2+1)).
d = (2+ b++); alert(d); // 4

вывод 4: переменная b содержит 2. так как ++ стоят после переменной, сначала выполнится 2 + b, это значение (4) присвоится переменной d, а затем уже выполнится b++
alert(a); // 3
alert(b); // 3

обе переменные a и b были увеличены на 2, соответственно их значение теперь 3.
